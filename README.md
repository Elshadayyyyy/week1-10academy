# week 1 task: Predicting Price Moves with News Sentiment

## Project Overview
 explores the relationship between news sentiment and stock price movements. It consists of three main tasks:

1. **Task 1:** Git and GitHub setup with Exploratory Data Analysis (EDA)  
2. **Task 2:** Quantitative analysis using PyNance and TA-Lib  
3. **Task 3:** Correlation analysis between news sentiment and stock movements
